# 基于多层感知机的 MNIST 手写数字识别实验报告

## 1. 实验目的

本实验旨在参考课程提供的学习资料，构建一个多层感知机（MLP）神经网络，在 MNIST 手写数字数据集上完成训练与测试，并给出训练过程、推理过程和实验结果。

通过本实验，希望进一步理解以下内容：

1. 感知机与多层感知机的基本原理；
2. 激活函数在非线性建模中的作用；
3. 前向传播、损失函数、反向传播和参数更新的基本流程；
4. MLP 在图像分类任务中的能力与局限性。

---

## 2. 理论基础

### 2.1 感知机与多层感知机

单层感知机本质上是一个线性分类器，只能处理线性可分问题。而手写数字分类属于更复杂的非线性问题，因此单层感知机无法很好地完成该任务。

多层感知机（MLP）通过引入隐藏层和非线性激活函数，使模型能够学习更加复杂的非线性映射关系，从而具备更强的表达能力。

### 2.2 激活函数的作用

如果网络中只有线性变换，即使堆叠多层，其整体仍然等价于一个线性模型。因此，必须通过激活函数引入非线性，才能使网络具备拟合复杂函数的能力。

本实验中采用 ReLU 作为隐藏层激活函数。其优点包括：
- 计算简单；
- 有利于梯度传播；
- 在深层网络中通常表现较好。

### 2.3 多分类问题

MNIST 手写数字识别任务是一个 10 分类问题，类别为 0 到 9。模型输出为 10 个类别对应的分数，最终取最大值对应的类别作为预测结果。

---

## 3. 数据集与预处理

### 3.1 数据集介绍

本实验使用 MNIST 手写数字数据集。该数据集包含大量 28×28 的灰度图像，每张图像对应一个数字标签（0~9）。

### 3.2 数据预处理方法

为了使数据能够输入 MLP 网络，本实验进行了如下处理：

1. 对图像像素值进行归一化；
2. 将原始 28×28 的二维图像拉平成 784 维向量；
3. 将标签作为分类任务的目标输出。

这种处理方式可以直接将图像输入全连接网络，但也会丢失图像原有的二维空间结构。

---

## 4. 模型设计

### 4.1 网络结构

本实验构建的 MLP 网络结构如下：

- 输入层：784
- 隐藏层1：256
- 隐藏层2：128
- 输出层：10

其中：
- 输入层对应 flatten 之后的图像向量；
- 两个隐藏层负责非线性特征变换；
- 输出层用于输出 10 个类别的分类结果。

### 4.2 损失函数与优化器

- 损失函数：CrossEntropyLoss
- 优化器：Adam

交叉熵损失适合多分类任务，Adam 优化器具有较好的收敛性能和训练稳定性。

---

## 5. 训练过程

训练过程主要包括以下步骤：

1. 从训练集读取一个 batch 的样本；
2. 将输入数据送入 MLP 网络，得到输出结果；
3. 根据模型输出与真实标签计算损失函数；
4. 通过反向传播计算各层参数梯度；
5. 使用优化器更新参数；
6. 重复以上步骤直至完成所有 epoch。

训练过程中重点记录：
- 每个 epoch 的训练损失；
- 每个 epoch 的测试准确率。

### 6.1 定量结果

### 6.2 可视化结果

这些结果可以帮助观察模型在具体样本上的识别效果。

## 7. 分析与讨论

### 7.1 MLP 的优点

- 结构简单，易于实现；
- 能够通过隐藏层和激活函数学习非线性映射；
- 适合作为入门级神经网络模型。

### 7.2 MLP 的局限性

虽然 MLP 能够完成 MNIST 手写数字识别任务，但它对图像的处理方式存在明显局限：

1. 将二维图像拉平成一维向量后，丢失了空间结构信息；
2. 全连接层参数量较大；
3. 对平移、缩放等图像变化的鲁棒性较弱；
4. 对图像局部特征的利用不如卷积神经网络自然。

因此，MLP 可以作为图像分类的基础模型，但不是处理图像任务的最终方案。

### 7.3 对后续模型的理解

从发展角度来看，MLP 是现代神经网络的重要雏形，而卷积神经网络（CNN）则是在图像任务上对 MLP 的进一步改进。CNN 通过局部连接和权值共享，更适合处理图像这种具有空间结构的数据。
